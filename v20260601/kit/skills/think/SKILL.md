## Quick Reference

**Research:** Never single-source critical findings — triangulate 2+ independent sources. Heavy research (5+ sources) → delegate to librarian agent.

**Reasoning:** For complex problems, decompose → solve with confidence (0.0–1.0) → verify → synthesize → reflect. If overall confidence < 0.8, identify the weakest link and retry or state the gap.

**Context budget (always apply):**

| Level | Action |
|-------|--------|
| <50% | Continue |
| 50-70% | Delegate to librarian |
| 70-85% | `/compact` or commit |
| >85% | Commit, start new thread |
