# Claude Code Harness Blueprint

Self-bootstrap a goal-centric, autonomously orchestrated Claude Code harness for any project.

## Quick start

1. Download `BLUEPRINT.md` from this repo
2. Open Claude Code in any directory
3. Tell Claude: "Read this blueprint and install the harness" — then hand it the file

The blueprint is addressed directly to Claude. It handles the Understanding Phase (reading your existing setup), installs the seed skills and hooks, extends your CLAUDE.md, and gives you a running harness in one session.

## Versions

Tagged releases correspond to versioned blueprint snapshots. The latest is always at `main`.

## What you get

- **12 seed skills** (discipline + universal build primitives)
- **9 hooks** (session continuity, quality gates, resilience)
- **CLAUDE.md discipline block** (8 principles, 5-phase task cycle, naming conventions)
- **A work queue** (`ops--backlog`) — cross-project source of truth for what's next
- **Autonomous orchestration** (`ops--swarm`) — parallel multi-agent builds
- **Session continuity** — handoff + live-state so no session opens from zero

See `BLUEPRINT.md` for the full instruction set.
